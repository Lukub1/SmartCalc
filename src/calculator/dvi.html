SmartCalc

Функции программы:

Калькулятор
Построение графиков
Расчет кредита
Расчет вкладов

Графический пользовательский интерфейс калькулятора содержит:

Кнопки от 0 до 9
Кнопки математических операторов (+,-,*,/,^), оператор остатка деления
Кнопки математических функций
Кнопку для выбора других режиимов
Поле для ввода выражания и вывода результатов
Поле для ввода переменной Х

Графический пользовательский интерфейс графика содержит:

Поле для ввода функции
Два поля для ввода промежутка, на котором отрисуется график
Поле для отрисовки графика

Графический пользовательский интерфейс расчета кредита содержит:

Поля для ввода суммы кредита, ставки, срока
Две кнопки для выбора типа кредита
Кнопку для расчета
Поля с расчетами переплат и общей выплаты
Таблицу с расчетом платежей

Графический пользовательский интерфейс расчета депозита содержит:

Поля для ввода суммы вклада, ставки, срока, налоговой ставки
Кнопку для расчета
Поле с расчетом дохода
